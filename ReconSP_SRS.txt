# ReconSP — Software Requirements Specification (SRS)

Version: 1.0
Tool Name: ReconSP
Type: Automated Reconnaissance & Attack Surface Intelligence Tool
Primary Users: Cybersecurity Analysts, Bug Bounty Researchers

---

# 1. SYSTEM OVERVIEW

ReconSP is an **automated reconnaissance orchestration tool** designed to discover and analyze the attack surface of a web target.

The system will perform:

* Infrastructure discovery
* Recursive subdomain enumeration
* Endpoint discovery
* JavaScript intelligence extraction
* Secret detection
* Cloud asset discovery
* Subdomain takeover detection
* Backup file discovery
* API mapping
* Technology fingerprinting
* Vulnerability scanning
* Attack surface prioritization
* HTML reporting

The objective is to convert **raw reconnaissance data into actionable intelligence**.

---

# 2. USER INPUT

ReconSP must accept only **two parameters**.

Required parameters:

```
-u --url
-d --depth
```

Example:

```
reconsp -u example.com -d 3
```

Explanation:

| Parameter | Description               |
| --------- | ------------------------- |
| -u        | Target domain             |
| -d        | Recursive subdomain depth |

Example depth behavior:

```
Depth 1:
example.com
api.example.com

Depth 2:
api.dev.example.com
stage.api.example.com

Depth 3:
internal.dev.api.example.com
```
